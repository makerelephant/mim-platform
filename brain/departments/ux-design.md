# Department: UX & Design

## Purpose
Defines how In Motion prioritizes and routes UX and design activity.

## CEO Role
The CEO is the designer and brand owner. Controls brand, marks, icons, patterns, and designed components directly.

## Ownership Model

| Signal Type | Meaning |
|-------------|---------|
| Brand / logo / icons | CEO owns directly |
| Components / patterns | CEO designs or approves |
| UI / UX design | CEO designs or guides |
| Design review | CEO approves |
| UX feedback | CEO advises |
| Product feedback | Product team handles |

## Routing Rules

| Topic | Send To |
|-------|--------|
| Feature UI design | Product designers |
| Implementation questions | Engineering |
| UX research requests | UX researchers |
| Design system work | Design team |
| Marketing visuals | Brand / marketing designers |
| Website design | Marketing or web team |
| Customer UX issues | Product + design |
| Product direction | Product leadership |

## Priority Hierarchy

```
Product usability
  ↓
Design system / components
  ↓
Brand integrity
  ↓
Feature design
  ↓
Visual assets
  ↓
Marketing visuals
  ↓
Minor UI tweaks
```

## Importance Model

### High

| Situation | Signals | Why |
|-----------|---------|-----|
| Core product UI decisions | Component change, design system update, UI pattern, layout redesign | Influences entire product experience |
| Product launch / feature design | Launch design, feature UI, product release | Tied to shipping features |
| Brand integrity | Logo usage, brand guidelines, brand compliance | Maintains external consistency |
| Major UX problems | Usability issue, user confusion, UX problem | Direct customer experience impact |

### Medium

| Situation | Signals |
|-----------|---------|
| Visual assets | Icons, illustrations, graphics |
| Marketing design | Landing page, campaign visuals |
| Design reviews | Design review, feedback requested, critique |

### Low

| Situation | Signals |
|-----------|---------|
| Minor visual tweaks | Spacing, color change, small UI tweak |
| General feedback | Opinion on layout, design suggestion |
| Informational | FYI, design inspiration, sharing updates |
