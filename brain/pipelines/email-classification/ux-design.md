# Email Classification Rule: UX & Design

## Category ID
`ux_design`

## Inbound From
- Product managers (feature design, product flows, usability)
- Engineers (design specs, UI clarification, handoff)
- Marketing team (brand assets, landing pages, campaign visuals)
- Customer success / support (UX issues, user confusion, feedback)
- Leadership (product direction, brand direction, major redesigns)

## Outbound To
- Engineering (implementation questions)
- Product leadership (product direction)
- Marketing / brand designers (marketing visuals, website)
- Product designers (feature UI)

## Note
CEO is the designer and brand owner. Strongest signals are brand, visual assets, components, patterns, design review, UI decisions. If the email involves how the product looks or behaves visually, it's likely this category.

## Signals

### Brand / Identity
Logo, brand, brand guidelines, brand usage, visual identity, brand update, brand assets, brand consistency, visual standard, design language, brand compliance

### Visual Assets
Icons, illustrations, graphics, visual assets, design assets

### UI / Product Design
UI, component, design spec, design system, layout, interaction, UX flow, design token, UI pattern, component library

### Design Review / Approval
Design review, feedback on design, approve design, sign off, design critique

### UX Feedback
UX feedback, user experience, usability issue, user confusion, interaction improvement, UX problem, customer feedback about UI

### Heuristic
- If it involves **how the product looks or behaves visually** → UX & Design
- If it involves **what the product should do** → Product / Engineering

## Importance

### High
- Core product UI / component decisions: component change, design system update, UI pattern decision, layout redesign
- Product launch / feature design: launch design, feature UI, product release, design needed for feature
- Brand integrity: logo usage, brand guidelines, brand compliance, visual identity approval
- Major UX problems: usability issue, user confusion, UX problem, customer feedback about UI

Priority order: product usability → design system / components → brand integrity → feature design

### Medium
- New icons / visual assets: icons, illustrations, graphics
- Landing pages / marketing design: landing page, campaign visuals, marketing design
- Design reviews: design review, feedback requested, design critique

### Low
- Minor visual tweaks: spacing adjustment, color change, small UI tweak
- General design feedback: thoughts on design, opinion on layout, design suggestion
- Informational threads: FYI, sharing design update, design inspiration
